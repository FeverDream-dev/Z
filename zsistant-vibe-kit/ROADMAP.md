# Zsistant Roadmap

## Now

- Use this kit to generate the actual implementation in OpenCode.
- Build phases 00–06 before connecting real channels.

## MVP

- `zazi` CLI.
- Local daemon and web UI.
- Agent create/list/chat.
- Isolated workspaces.
- Job status timeline.
- Telegram/Discord/WhatsApp dry-run adapters.
- LLM router with mock provider fallback.
- Persona trainer patch proposals.
- Inter-agent ACL bus.
- Skill analyzer.

## Beta

- Real channel setups.
- SQLite persistence.
- GitHub release workflow.
- Chrome MCP UI test workflow.
- Stronger provider integrations.
- Import OpenClaw Markdown migration summaries.

## Later

- Email connectors.
- Browser automation for end users.
- Sandboxed external skill execution.
- Paid cloud backend.
- Hosted model subscription.
- Multi-machine control plane.
