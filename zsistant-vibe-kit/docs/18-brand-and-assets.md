# 18 — Brand and Asset Notes

## Brand name

Zsistant

## CLI name

zazi

## Visual concept

A sharp **Z** inside a local control node. The dots around it represent many isolated agents coordinated by one lightweight daemon.

## Colors in generated placeholders

The included assets use a dark blue/purple control-tower style with cyan accents. These are placeholders and can be replaced by a designer later.

## Asset folders


## Important legal note

The channel icons in this pack are **generic placeholders**, not official Telegram, Discord, WhatsApp, GitHub, OpenCode, or OpenClaw logos. Replace them later only if you follow each service's brand guidelines.
