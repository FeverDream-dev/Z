# Release Checklist

- [ ] Tests pass.
- [ ] Manual web UI smoke test done.
- [ ] CLI smoke test done.
- [ ] No secrets in logs or repo.
- [ ] Changelog drafted.
- [ ] Migration notes updated if needed.
- [ ] Security notes updated if needed.
- [ ] Human explicitly approved push/tag/release.

Approval phrase:

```text
Yes, push and release vX.Y.Z.
```
