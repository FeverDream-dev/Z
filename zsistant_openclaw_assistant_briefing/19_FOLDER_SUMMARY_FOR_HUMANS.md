# Folder Summary for Humans

This folder exists because the coding AI kept building a normal ChatGPT clone.

The documents explain that Zsistant is supposed to be an assistant platform.

The biggest ideas are:

1. Assistants are first-class objects.
2. Chat is only one surface.
3. Assistants need tools, channels, memory, knowledge, jobs, permissions, and logs.
4. Chrome/MCP/browser automation is central.
5. Telegram/WhatsApp/Discord/Slack-style channels matter.
6. Developer Mode is a real product feature.
7. Scheduled jobs and chronological activity make assistants useful across time.
8. The UI must be simple and beautiful, but honest.
9. No fake integrations or placeholder data.
10. Zsistant should feel like a control center for AI workers.
