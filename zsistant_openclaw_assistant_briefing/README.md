# Zsistant / zazi: Assistant-First Product Briefing

This folder is written for a coding AI that keeps misunderstanding the product and reducing it to a simple ChatGPT clone.

Read this folder before editing code.

The product is not "a chat app". It is an assistant platform.

A chat app waits for text and replies with text.
An assistant is an always-available operator with memory, tools, permissions, channels, jobs, browser access, files, context, workflows, and multiple specialized personalities.

## What this folder intentionally avoids

This folder does not tell you which programming language, framework, ORM, or CSS library to use. It is not an implementation recipe. It explains the product idea and expected behavior so the coding AI stops building the wrong thing.

## One sentence summary

Zsistant is a personal/team AI operating layer where the user creates and manages many real assistants that can communicate through multiple channels, use tools, browse through Chrome/MCP, remember context, run scheduled jobs, coordinate with other agents, and expose advanced developer controls when needed.
